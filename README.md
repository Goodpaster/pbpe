Installation
------------

* Prerequisites
    - Python 3.5
    - Numpy 1.8.0 or higher
    - Scipy 0.10 or higher
    - h5py 2.3.0 or higher
    - PySCF 1.4 or higher
    - input_reader 1.2.2 or higher
    - Cython 0.23.4 or higher

* Compile modules

        make

* To clean/delete compiled modules

        make clean
